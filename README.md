 # Project Title
Todo List with User Authentication using React JS

## Description
A managment list for tasks 
 
### Dependencies
* REACT JS 18.3.1 
* Bootstrap 5.3.3

### Installing
* NODE v22.1.0
 
### Executing program

* Open workspace in Visual Studio Code
* Open terminal in VS and change directory to workspace folder
* Use the command "npm start"
